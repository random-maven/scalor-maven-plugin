scalor:scala-js-env-conf-phantomjs

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.2.3.20180215190318:scala-js-env-conf-phantomjs

Description:

Configure Scala.js JavaScript VM environment for testing: provide scala-js-junit-tools settings for Phantom.js.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-resources.
