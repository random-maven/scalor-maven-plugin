scalor:scala-js-link

Full name:

com.carrotgarden.maven:scalor-maven-plugin_2.12:1.3.2.20180223212324:scala-js-link

Description:

Generate Scala.js runtime JavaScript for all scopes. Invokes goals: scala-js-link-*.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-classes.
